Metamask Integration

Before adding the Monomer L1 Devnet to your Metamask wallet, please ensure that you perform the following steps:

  • Ensure that you use a separate test wallet for the Monomer L1 Devnet.
  • If the Monomer L1 Devnet wallet was previously configured, remove the network from Metamask and click the "Clear activity and nonce data" button in the Metamask "Advanced" settings.
  • Run the Monomer appchain with the flags to start the L1 devnet and OP Stack:
    • ./{AppName}d monomer start --minimum-gas-prices 0.01wei --api.enable --monomer.dev-start --monomer.dev.l1-user-address "0x{YourL1WalletAddress}"
